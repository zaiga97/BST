Detailed Description

template<typename Tkey, typename Tvalue, typename Tcompare = std::less<Tkey>>
template<typename Tpair_i>
class BST< Tkey, Tvalue, Tcompare >::_iterator< Tpair_i >

_iterator is a templated class used for the pythonic for loop syntax. It only contains a raw pointer to a node as a private member and several operators to move in the tree and compare/extract informations from the node it's pointing at. The common use is to travers the tree from the smallest key node to the biggest one.

template<typename Tpair_i >
BST< Tkey, Tvalue, Tcompare >::_iterator< Tpair_i > & BST< Tkey, Tvalue, Tcompare >::_iterator< Tpair_i >::operator++

This pre-increment operator is the core of the _iterator class and it's responsable for the traversing of the tree. All it does is changing the node the iterator is pointing to the one in the tree with the next larger key. If such a node it's not present, (it's pointing to the node with biggest key in the tree), retuns an iterator pointing to a nullptr.
